这个博客主要用来记录三类内容：

- 读书笔记
- 论文阅读笔记
- RL / LLM 训练中的实现备忘与阶段性理解

它更像一个可长期维护的知识索引，而不是纯时间流博客。

## 为什么要单独做一份博客

很多训练相关的知识并不是“看过就会”的内容，而是需要反复回查的：

- 一套算法推导为什么成立
- 某个训练 recipe 的关键超参数是什么
- 某篇论文到底给了什么可复用的结论
- 某次实验踩过的坑后来有没有复现价值

所以这里会优先追求：

- 短标题，方便快速定位
- 稳定链接，方便跨设备查阅
- 标签化组织，而不是只按时间堆叠
- 搜索友好，方便以后自己回看

## 目前关注的主题

- PPO / GRPO / DPO 等后训练方法
- reward model 与 preference data
- reasoning model 的训练路径
- RL 与 LLM 训练之间可以互相借鉴的部分
- 适合长期复用的实验记录模板

## 使用方式

如果你以后想快速回看某个主题，建议优先使用：

- `专题 / 标签`：按方法或主题跳转
- `归档`：按时间回顾最近在集中读什么
- `搜索`：适合查术语、模型名、论文名
- `阅读清单`：维护未来要读和已经读完的材料
